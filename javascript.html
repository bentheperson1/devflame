JavaScript Course


For Game Development, JavaScript is most commonly used for things like servers and HTML5 games.

In this course, we will make a game using the Phaser.JS Game Framework that will be instantly sharable on the web.


Note: This course assumes you know at least the very basics of JavaScript.
If you don't know JavaScript, We would recommend freeCodeCamp's Basic JavaScript Series.




Lessons:

  • Setting up the development environment
  • Getting Phaser.JS & Course Assets
  • Testing Everything
  • 1945: Setting up
  • 1945: Adding the player
