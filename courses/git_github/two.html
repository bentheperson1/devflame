Git + Github Course


Getting our code to Github

  • First, Open a Git Bash in your project folder as shown in the previous lesson.

  • Next, Go to Github.com and create an account.

  • Next, Click "New" on the repositories section, Put in the name of your project, and then select if you would like it to be public or private.
    This is where we will host all of our code.

  • Next, type git config --global user.name "your_username" and git config --global user.email "your_email_address@example.com"
    But make sure to replace "your_username" with your actual GitHub username and "your_email_address@example.com" with your actual GitHub email.

    If you did this correctly, you will have just logged into GitHub using the Git Bash.

  • Next, type git init. This will initialize a new Git folder which will allow you to send and recieve files from GitHub.

  • Then, type git add .. This will add everything to a Git package which you can then commit and push to Github.

  • After that, type git commit -m "your commit message". This will add commit your Git package and make it able to be pushed to Github.

  • Finally, copy & paste the commands from Github under the section "push an existing repository from the command line".
    This will add the end point to your repository which you created earlier.

  • If you did everything correctly, you should be able to refresh your GitHub repository page and see all of your folders & code.
