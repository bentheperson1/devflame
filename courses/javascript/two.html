Getting Phaser.JS & Course Assets

  • First, we are going to need to add the Phaser.JS script to our HTML file, So open up Visual Studio Code, create a new file called "index.html" inside your project folder inside the htdocs folder of xampp, and then type ! and then press TAB. This should create boilerplate code for you to get started with.


  • If for whatever reason boilerplate code isn't created, you can copy & paste the following code: 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Game</title>
</head>
<body>
    
</body>
</html>

  • Next, add in the following line of code into the <head> element: 

    <script src="//cdn.jsdelivr.net/npm/phaser@3.52.0/dist/phaser.min.js"></script>

    We now have added Phaser.JS into our project.

  • Finally, download this zip file including all of the art, music, and code for this course.
