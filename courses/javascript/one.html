Setting Up the Development Environment

In this lesson we will be setting up the development environment to allow for JavaScript development.

  • First, we are going to need a local server to serve us javascript content.
    I personally use XAMPP, So that is what we are going to be using in this course.

  • Second, we are going to need some kind of text/code editor.
    We are going to be using Visual Studio Code for this course because it is not just limited to JavaScript. It has support for pretty much all languages.

  • There are tutorials on how to setup both of these pieces of software on their respective websites so we will not be covering it here.
